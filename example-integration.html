📦 How to use

  1. Build the web component once:
    npx vite build --config vite.lib.config.js
  2. Include dist/virtual-try-on.js on your page
  3. Place <virtual-try-on> anywhere in your HTML
  4. That's it — no framework required, no extra setup

✨ Features

  • Real-time face tracking via MediaPipe
  • Foundation, blush, lipstick, lip liner, gloss
  • Auto foundation shade matching
  • Skin smoothing (WASM)
  • Works on mobile & desktop
  • Zero CSS conflicts — fully encapsulated

🔧 Configuration via attributes

<virtual-try-on foundation-product-sku="212-66160" blush-product-sku="212-15505" lipstick-product-sku="212-12703" lipliner-product-sku="212-519050" ></virtual-try-on>

⚠️ Requirements

  • Page must be served over HTTPS (or localhost)
  • Browser must support getUserMedia (camera access)
  • Allow camera permission when prompted
  • The widget manages its own lifecycle — mount/unmount is automatic
